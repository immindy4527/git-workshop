# 上課心得   
我那天晚到了，快步往樓梯口方向時已聽見老師麥克風宏亮的聲音，因此我有些緊張老師是不是一個嚴格的人，但上來後感覺老師反而是口語輕鬆又活潑地講述自己的事情，可惜我遲到沒聽到全部，但至少不是開始上正課，不然我會更慌張緊繃。   

從開始正式"寫程式"課程後，腳步就亂了許多，什麼邏輯概念，如何向電腦下指令，那是我以前沒有學過的，即是我高中是理組(同學知道後很驚訝)，且我身體和腦袋記憶力因為生過病而不如從前。開始懷疑自己當初為什麼要來報名這堂課，甚至暗自告訴自己，即是 JS、PHP 函數...那些很差，那就退一步當個網頁 UI 設計師也好，大學和剛出社會時是設計的，只是我變成是網頁設計師，薪水不會比過去少，做的事也比較單純一些；但我也依然很掙扎自己是否可以再撐住學好，即使學得比大家慢，也要成為一個厲害的工程師、成為一個專業的人。   

回到4/30老師上課，其實我當天狀況很不好，所以我吸收程度又比之前上 PHP 的時候差。但我依然盡力去聽，老師其實講得很好，至少我在很清醒時是聽得懂老師講解 git 操作 (偷偷說，比學姊講得好，共筆真的很棒)，但我真的很慚愧自己為什麼昨天不早點睡不穿多一點衣服再出門，所以我聽過後又忘了。   
然後到下午我就快陣亡了，但至少我有概念了，只是有一些路徑資料夾我容易搞混，不知道怎麼打指令找，可能還需要老師未來能夠慢下來提點我。   

我很開心能遇到一個很活潑熱情的老師，在大家煩悶鑽研程式的時候增添了活絡的氣氛，我也比較沒那麼害怕學習，希望老師未來更能夠教我們每種 code 寫法，應用在不同功能畫面。
